# zhugz-skills

> 一個為「想得更深、動得更快」而生的 AI Skill 庫

---

## 這是什麼？

這是 **zhu.gz** 整理的個人 AI Skill 合集，專為 [Google AI Edge Gallery](https://github.com/google-ai-edge/gallery) 設計。

你不需要懂程式碼，也不需要是工程師。

只要有手機、下載好 App、匯入 Skill，就能讓 AI 用你想要的方式思考。

---

## 為什麼會有這個庫？

我是一個：

- 🧠 **喜歡深度思考** ，但容易在腦袋裡打轉，說不清楚
- ⚡ **行動派** ，但有時卡在某個環節就停下來了
- 🔍 **好奇心很強** ，什麼都想試，什麼都想懂
- 📐 **追求結構和效率** ，不喜歡模糊的答案

普通 AI 對話很容易變成「你問、它答」的模式，幫不了你真正想清楚一件事。

所以我開始把自己的思考方式「寫進」AI，讓它成為真正的思維夥伴，而不是答題機器。

這個庫，就是這些嘗試的累積。

## Skill 一覽

### 🧠 思維與決策

| Skill              | 用途                                       |
| ------------------ | ------------------------------------------ |
| `thinking-partner` | 結構化思維夥伴，幫你把問題想透、決策想清楚 |

**適合什麼時候用：**

- 腦袋打結，不知道從哪裡開始

### 🎨 設計思維

| Skill            | 用途                                     |
| ---------------- | ---------------------------------------- |
| `design-partner` | 設計師專用思維夥伴，從模糊想法到清晰提案 |

**適合什麼時候用：**

- 腦袋有感覺但說不出方向
- 需要有人直接告訴你這個設計哪裡不對
- 客戶一直改稿，不知道問題出在哪
- 要跟客戶或非設計師解釋「為什麼這樣設計」
- 需要寫提案、簡報、設計說明

**涵蓋範疇：** 品牌／視覺設計・UI／產品設計・空間／展覽設計

## 怎麼使用？

### 第一步：下載 App

- iOS：[App Store 搜尋 Google AI Edge Gallery](https://apps.apple.com/us/app/google-ai-edge-gallery/id6749645337)
- Android：[Google Play 搜尋 Google AI Edge Gallery](https://play.google.com/store/apps/details?id=com.google.ai.edge.gallery)

### 第二步：下載模型

打開 App 後，下載任一語言模型（推薦 **Gemma-4-E2B** ，免費、可離線）

### 第三步：匯入 Skill

1. 進入 **Agent Skills** 頁面
2. 點擊 **Skills** → **+**
3. 貼上你想要的 Skill 的 Raw URL（見下方）
4. 匯入完成，開始使用

## 給完全零基礎的你

**什麼是 Skill？**

就像是給 AI 裝一個「模式」。普通 AI 什麼都能回答，但沒有固定風格。裝了 Skill，它就會用特定的方式幫你思考。

**需要付費嗎？**

不需要。App 免費、模型免費、這個 Skill 庫也免費。

而且所有運算都在你的手機上完成，不會上傳任何資料。

**支援哪些 App？**

目前專為 Google AI Edge Gallery 設計。概念上也適用於其他支援自訂 System Prompt 的 AI App。
